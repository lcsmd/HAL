===============================================
START QM VOICE LISTENER
===============================================

In the QM window that's open, type:

PHANTOM VOICE.LISTENER

Then press Enter.

You should see:
- HAL Voice Listener starting...
- Port: 8767
- Voice Listener active on port 8767
- Waiting for connections...

If the QM window is closed, open a new one:
1. cd C:\qmsys\bin
2. qm -account HAL
3. PHANTOM VOICE.LISTENER
