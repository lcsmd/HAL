EXIT.PACKAGE()

    

 

The EXIT.PACKAGE() function exits from a software package.

 

 

Format

 

EXIT.PACKAGE(package.name)

 

where

package.nameis the name of the package.

 

 

This function returns True if it is successful, False if it fails. In the event of failure, the STATUS() function can be used to determine the cause.

 

 

See also:
Package licensing, ENTER.PACKAGE(), PACKAGE.LICENCE, !PACKAGE()
