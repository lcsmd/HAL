LOGMSG

    

 

The LOGMSG command adds a line to the system error log.

 

 

Format

 

LOGMSG text

 

where

 

textis the message to be logged.

 

 

This command is identical in effect to use of the LOGMSG statement in a QMBasic program.

 

QM includes the option to maintain a log of system error messages in a file named errlog in the QMSYS account. The LOGMSG command can be used to write messages into this file. If the error log is disabled by setting the ERRLOG configuration parameter to zero, the LOGMSG command will be ignored.
