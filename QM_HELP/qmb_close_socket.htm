CLOSE.SOCKET

    

 

The CLOSE.SOCKET statement closes a socket.

 

 

Format

 

CLOSE.SOCKET skt

 

where

 

sktis the socket variable corresponding to the socket to be closed.

 

 

The socket previously opened using ACCEPT.SOCKET.CONNECTION(), CREATE.SERVER.SOCKET() or OPEN.SOCKET() is closed. The skt variable is set to unassigned.
