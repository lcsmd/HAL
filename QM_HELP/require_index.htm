REQUIRE.INDEX

    

 

The REQUIRE.INDEX keyword causes the query processor to terminate the query unless it can make use of an alternate key index.

 

 

Format

 

REQUIRE.INDEX

 

 

Use of the REQUIRE.INDEX keyword can help determine whether a query has been phrased in a manner that can make use of an alternate key index. If it cannot, a diagnostic message explaining the reason is displayed.

 

This keyword would typically only be used in a development environment.
