DELETELIST

    

 

The DELETELIST statement deletes a select list from the $SAVEDLISTS file.

 

 

Format

 

DELETELIST name

 

where

 

nameis the name of the $SAVEDLISTS entry to be deleted.

 

 

The DELETELIST statement deletes the previously saved select list identified by name from the $SAVEDLISTS file. No error occurs if the list does not exist.
