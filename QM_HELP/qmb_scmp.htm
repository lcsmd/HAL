SCMP()

    

 

The SCMP() function performs comparison of two integer numeric strings of any length.

 

 

Format

 

SCMP(a, b)

 

where

 

a, bare numeric strings.

 

 

The values passed in as arguments a and b must be strings that contain only digits. A leading minus sign is allowed.

 

The returned value will be

-1a is less than b
0a is equal to b
1a is greater than b
