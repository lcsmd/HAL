T.DET, T.EOD, T.FWD, T.RDLBL, T.READ, T.REW, T.STAT, T.WEOF

    

 

The T.xxx utility commands perform various functions to a tape device.

 

 

Format

 

T.DETDetach device
T.EODPosition to end of data
T.FWDSkip forwards one file
T.RDLBLRead tape label
T.READRead tape
T.REWRewind tape
T.STATReport device status
T.WEOFWrite end of file marker

 

The tape or pseudo-tape to be processed must first be assigned using the SET.DEVICE command.
