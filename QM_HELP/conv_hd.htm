HTML Data Conversion (HD)

    

 

The HD conversion code translates HTML reserved characters to/from their escape codes.

 

Used as an output conversion, the minimum set of reserved characters are replaced with their HTML escape codes. These are

&&amp;
<&lt;
>&gt;
"&quot;
'&#39;

 

In addition, character 128 is converted to &nbsp; (non-breaking space) and the internal representation of the Euro currency symbol is converted to &euro; The Unicode character set defines the Euro symbol as codepoint 0x20AC and the ECS version of QM supports this. On a non-ECS system, the commonly adopted but not strictly defined use of character 160 as the Euro symbol is adopted.

 

Used as an input conversion, the full set of HTML escape codes is recognised.
