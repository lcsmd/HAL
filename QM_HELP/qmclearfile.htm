QMClearFile()

    

 

The QMClearFile() function deletes all content from a file. It is analogous to the QMBasic CLEARFILE statement.

 

The function takes a single argument, FileNo, as the file number returned from a previous use of the QMOpen() function.

 

The code fragment in the examples below opens the CLIENTS file and clears it.

 

 

C

void QMClearFile(int FileNo)

fClients = QMOpen("CLIENTS");

QMClearFile(fClients);

 

VB.Net

QMClearFile(ByVal FileNo as Integer)

fClients = QMOpen("CLIENTS")

QMClearFile(fClients)

 

QMBasic Class Module

ClearFile(FileNo)

fClients = session->Open("CLIENTS")

session->ClearFile(fClients)

 

Java

ClearFile(int FileNo)

fClients = qm.Open("CLIENTS");

qm.ClearFile(fClients);

 

Python

ClearFile(FileNo)

fClients = qm.Open("CLIENTS")

qm.ClearFile(fClients)
