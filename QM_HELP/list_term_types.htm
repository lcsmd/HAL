LIST.TERM.TYPES

    

 

The LIST.TERM.TYPES command displays a list of terminal types.

 

 

Format

 

LIST.TERM.TYPES

 

 

The LIST.TERM.TYPES command scans the terminfo database and displays an alphabetically sorted list of available terminal types and their descriptions.

 

 

Example

 

LIST.TERM.TYPES

ANSI              ANSI/pc-term compatible with color

ANSI-GENERIC      Generic ANSI standard terminal

DEC-VT100         VT100 emulation

DEC-VT220         DOS tnvt200 terminal emulator

DEC-VT330         DEC vt340 graphics terminal with 24 line page

DEC-VT340         DEC vt340 graphics terminal with 24 line page

DEC-VT400         DEC vt400 24x80 column autowrap

etc.....
