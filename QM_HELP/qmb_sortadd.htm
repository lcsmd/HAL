SORTADD

    

 

The SORTADD statement adds data to the sort system.

 

 

Format

 

SORTADD sortvar, {MAT} keys {, data}

 

where

 

sort.varis a sort variable created by a previous use of SORT().

 

keysis the name of a single dimensional array of sort keys prefixed with MAT. For a single key sort, keys may be specified as a scalar variable or an expression. In either case, the data type of this variable may change.

 

datais the optional data associated with the given keys. If omitted, this defaults to a null string. This value is ignored if the SRT$DATA flag was not set in the first key description passed to SORT().

 

 

The SORTADD statement adds the given keys and optional data to the sort session related to sortvar.
