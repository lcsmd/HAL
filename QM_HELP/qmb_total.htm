TOTAL()

    

 

The TOTAL() function accumulates totals for use with the CALC query processor keyword. It is only available in dictionary I-type items.

 

 

Format

 

TOTAL(expr)

 

where

 

expris an expression.

 

 

The TOTAL() function can be used in dictionary I-type expressions. While processing the detail lines of a report, the TOTAL() function returns the value of the expression but also accumulates a running total internally. When the query includes fields prefixed by the CALC keyword, the expression is re-evaluated on the total lines of the report using the accumulated value in place of the TOTAL() function.
