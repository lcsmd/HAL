REDO

    

 

The REDO command repeats a command at specified intervals.

 

 

Format

 

REDO {interval {count}} command

 

where

 

intervalis the delay in seconds between successive executions of command. If omitted or specified as zero, there is no delay.

 

countis the number of times command is to be executed. If omitted or specified as zero, the command is repeated forever.

 

commandis the command to be executed.

 

 

The REDO command repeats the specified command every interval seconds.

 

The REDO can be terminated before the count has expired by pressing any key on the keyboard.
