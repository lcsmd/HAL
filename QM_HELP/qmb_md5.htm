MD5()

    

 

The MD5() function returns the 32 digit hexadecimal message digest value for a given string.

 

 

Format

 

MD5(string)

 

where

 

stringis the string to be encoded.

 

 

The MD5() function provides a more comprehensive validation process than use of the default mode of the CHECKSUM() function. The returned value is a 32 digit hexadecimal string.

 

MD5 digest values are defined to work on byte level data. On an ECS mode system, it will be necessary to transform the data to a byte stream using the BS conversion code if it may contain characters outside the 8-bit set. Failure to do this will result in only the bottom 8 bits of each character in the source data being examined during encoding.

 

 

Example

 

DISPLAY MD5('The quick brown fox jumps over the lazy dog')

 

The above example would print '9e107d9d372bb6826bd81d3542a419d6'.
