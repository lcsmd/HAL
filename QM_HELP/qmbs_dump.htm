!DUMP()

    

 

The !DUMP() subroutine shows string data in hexadecimal and character form.

 

 

Format

 

CALL !DUMP(text {, lptr})

 

where

 

textis the string to be printed.

 

lptris the optional print unit number for the output. If omitted, the output is sent to the screen.

 

 

The !DUMP() subroutine is intended to be used as a diagnostic aid in programs that work with binary data. The text string is reported in the same form as use of the DUMP command to show the contents of a database record.

 

00000000: 43 61 72 74 77 72 69 67 68 74 2C 20 44 FE 37 20 | Cartwright, D^7

00000010: 53 70 72 69 6E 67 20 47 72 6F 76 65 FD 4E 6F 74 | Spring Grove]Not

00000020: 74 69 6E 67 68 61 6D FE 31 2D 31 FD 33 2D 31    | tingham^1-1]3-1

 

In an ECS mode system, if the data contains any characters outside the 8-bit set, each character is shown as four hexadecimal digits. The above example becomes

 

00000000: 0043 0061 0072 0074 0077 0072 0069 0067  | Cartwrig

00000008: 0068 0074 002C 0020 0044 00FE 0037 0020  | ht, D^7

00000010: 0053 0070 0072 0069 006E 0067 0020 0047  | Spring G

00000018: 0072 006F 0076 0065 00FD 004E 006F 0074  | rove]Not

00000020: 0074 0069 006E 0067 0068 0061 006D 00FE  | tingham^

00000028: 0031 002D 0031 00FD 0033 002D 0031       | 1-1]3-1
