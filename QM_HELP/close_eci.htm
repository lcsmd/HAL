CLOSE.ECI

    

 

The CLOSE.ECI command terminates all External Call Interface child processes.

 

 

Format

 

CLOSE.ECI

 

 

An External Call Interface handler normally continues to run until the QM process terminates or uses LOGTO to move to a different account. The CLOSE.ECI command terminates all ECI handler child processes, perhaps to allow the handler to be recompiled.
