QMWriteSeq()

    

 

The QMWriteSeq() function writes a line of text to a sequential file. This is analogous to the QMBasic WRITESEQ statement.

 

The function arguments are:

 

FileNois the file number returned by a previous QMOpenSeq() call.

 

Recis the data to be written.

 

The example program statements below write the line of text in ReportData to the file opened to the fReport variable.

 

 

C

void QMWriteSeq(int FileNo, char * Rec)

QMWriteSeq(fReport, ReportData);

 

VB.Net

QMWriteSeq ByVal FileNo as Integer, ByVal Rec as String

QMWriteSeq(fReport, ReportData)

 

QMBasic Class Module

WriteSeq(FileNo, Rec)

session->WriteSeq(fReport, ReportData)

 

Java

WriteSeq(int FileNo, String Rec)

qm.WriteSeq(fReport, ReportData);

 

Python

WriteSeq(FileNo, Rec)

qm.WriteSeq(fReport, ReportData)
